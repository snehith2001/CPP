	Exception Handling 
	==================
		What are exceptions?
			Exceptions are runtime abnormal conditions/ behavior that occurs when you program is executed.
			example: wrong input/files opening
			
		Handling of Exception?
			Throwing or raising an exception is done using throw keyword
			Handling of exceptions is done in try{}catch() block.
